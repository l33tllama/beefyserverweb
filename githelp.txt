HANDY GIT COMMANDS
For people who want to get stuff done, not learn a new thing..

REMOVE STUFF FROM CACHE (After adding stuff to .gitignore) 
git rm -r --cached .
git add .
git commit -m 'Removed crap'

ADD NEW STUFF
git add .

COMMIT YOURSELF BEFORE YOU WRECK YOURSELF
git commit -m 'Cure cancer. You're welcome.'

PUSH UP THE THROUGH PIPES
git push origin master
